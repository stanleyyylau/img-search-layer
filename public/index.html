Image search API service

image search API, super cool, just pass any string to https://st-imgsearch.herokuapp.com/api/search/ You get your response with image infos back in JSON format, you can also paginate the result through offset query.

Example useage

  • https://st-imgsearch.herokuapp.com/api/search/cats
  • https://st-imgsearch.herokuapp.com/api/search/cats?offset=20
  • You can also get the top 10 search queries with https://st-imgsearch.herokuapp.com/api/latest
